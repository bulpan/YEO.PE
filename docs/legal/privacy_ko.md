# YEO.PE 개인정보 처리방침

**최종 업데이트: 2026년 1월 8일**

YEO.PE(이하 "서비스")는 이용자의 개인정보를 중요시하며, 「개인정보 보호법」을 준수합니다.

---

## 1. 수집하는 개인정보

### 필수 수집 항목
| 항목 | 수집 목적 | 보유 기간 |
|------|----------|-----------|
| 이메일 주소 | 계정 식별, 로그인 | 회원 탈퇴 시 즉시 삭제 |
| 닉네임 | 서비스 내 표시명 | 회원 탈퇴 시 즉시 삭제 |
| 소셜 로그인 ID | Apple/Google/카카오/네이버 인증 | 회원 탈퇴 시 즉시 삭제 |

### 선택 수집 항목
| 항목 | 수집 목적 | 보유 기간 |
|------|----------|-----------|
| 프로필 이미지 | 채팅 시 표시 | 회원 탈퇴 시 즉시 삭제 |
| 푸시 알림 토큰 | 알림 전송 | 토큰 만료 또는 탈퇴 시 삭제 |
| 디바이스 정보 | 푸시 알림 최적화 | 탈퇴 시 삭제 |

### 자동 수집 항목
| 항목 | 수집 목적 | 보유 기간 |
|------|----------|-----------|
| 채팅 메시지/이미지 | 대화 기능 제공 | **생성 후 24시간 뒤 자동 삭제** |
| 채팅방 정보 | 실시간 소통 제공 | **생성 후 24시간 뒤 자동 삭제** |
| 신고 정보 | 서비스 악용 방지 | 처리 완료 후 1년 |
| 차단 정보 | 사용자 보호 | 해제 또는 탈퇴 시 삭제 |

---

## 2. 개인정보의 수집 방법
- 회원가입 시 이용자가 직접 입력
- 소셜 로그인(Apple, Google, 카카오, 네이버) 연동
- 앱 사용 중 자동 생성(메시지, 푸시 토큰 등)

---

## 3. 개인정보의 이용 목적
1. **서비스 제공**: 근거리 사용자 연결, 실시간 채팅
2. **계정 관리**: 회원 인증, 로그인, 알림 전송
3. **서비스 개선**: 오류 진단, 통계 분석(비식별 처리)
4. **안전한 환경**: 신고/차단 기능, 부정 이용 방지

---

## 4. 개인정보의 제3자 제공
서비스는 이용자의 동의 없이 개인정보를 제3자에게 제공하지 않습니다.
다만, 다음의 경우 예외로 합니다:
- 법령에 의한 수사기관의 요청
- 이용자가 사전에 동의한 경우

---

## 5. 개인정보의 처리위탁

| 위탁 업체 | 위탁 업무 |
|----------|----------|
| Oracle Cloud Infrastructure | 서버 호스팅, 데이터 저장 |
| Firebase (Google) | 푸시 알림 전송 |
| Object Storage | 이미지 저장 |

---

## 6. 이용자의 권리
이용자는 언제든지 다음을 요청할 수 있습니다:
1. **열람권**: 보유 중인 개인정보 확인
2. **정정권**: 잘못된 정보 수정
3. **삭제권**: 계정 삭제 시 모든 정보 즉시 삭제
4. **처리정지권**: 개인정보 처리 중단 요청

**요청 방법**: support@yeop3.com 또는 앱 내 [설정 > 계정 삭제]

---

## 7. 개인정보의 안전성 확보 조치
1. **암호화**: 비밀번호, 민감 정보 암호화 저장
2. **접근 제한**: 최소한의 담당자만 접근
3. **정기 점검**: 보안 취약점 정기 점검
4. **24시간 삭제**: 메시지/채팅방 자동 삭제로 정보 노출 최소화

---

## 8. 만 14세 미만 아동
서비스는 만 14세 미만 아동의 가입을 받지 않습니다.
만 14세 미만 아동의 개인정보가 수집된 것으로 확인될 경우 즉시 삭제합니다.
